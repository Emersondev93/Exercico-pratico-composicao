# 🛒 Sistema de Pedidos em Java

Sistema de pedidos em Java desenvolvido para revisar e consolidar conceitos fundamentais de POO e coleções.

## 🚀 Funcionalidades
- **Cadastro:** Clientes e pedidos.
- **Manipulação:** Adição de itens ao pedido.
- **Cálculos:** Valor total automático.
- **Controle:** Status do pedido via Enum.

## 🧠 Conceitos Praticados
- **POO:** Classes, objetos, encapsulamento e `toString()`.
- **Associações:** Composição de objetos.
- **Coleções:** `ArrayList` para itens.
- **Tipos & Datas:** Enums (`StatusPedido`), `LocalDate` e `LocalDateTime`.
- **Lógica:** Estruturas de repetição, validações e tratamento de exceções.

## 🛠️ Como Executar
1. Clone o repositório: `git clone https://github.com`
2. Abra o projeto na sua IDE favorita (Eclipse, IntelliJ, VS Code).
3. Execute a classe principal (que contém o método `main`).
